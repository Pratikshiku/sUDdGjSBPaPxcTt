## Clothing Production Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/55441a9d78db4b9eb3db0e9919e19594/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Personnel Arrangement
5. Samples
6. System Announcements
7. Personal Center
8. Attendance
9. Salary Management


**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. User Management
4. Personnel Arrangement Management
5. Attendance Management
6. Salary Management
7. Sample Management
8. Finished Product Inspection Management
9. Order Management
10. Raw Material Warehouse Management
11. Raw Material Inbound Management
12. Raw Material Outbound Management
13. Finished Garment Warehouse Management
14. Finished Garment Outbound Management
15. System Management (System Announcements, Carousel Management)
